Projects

Here are some of the projects I've worked on. Feel free to explore:

    Video Anomaly Detection

    This project is my attempt to reproduce the results of a paper from the CVPR conference called "Learning Normal Dynamics in Videos with Meta Prototype Network". See the paper here: https://arxiv.org/abs/2104.06689 I built and trained the base autoencoder with the Dynamic Prototype Unit (DPU). This project is a work in progress. I will test this model and build the few-shot Meta-Prototype Unit (MPU) in the future. I also wrote a critique based on this paper. This critique gives an in-depth overview of the theoretical aspects of the paper and points out some possible areas of improvement.

    Diabetic Retinopathy Detector Web App

    This web application allows the user to upload a retinal image of the eye, sends this image a neural network model, and the GPT 3.0 language model conveys the results to the user. See the report in the github repository for further details about the UI functionality and the neural network model training and results.

    Gravitational Potential of an Elliptical Mass Distribution

    In an Intermediate Mechanics course taken as credit towards a minor in physics, we were tasked to model the gravitational force produced by an elliptical mass distribution. This was a natural build off of what was discussed regarding modeling the gravitational force of various mass distributions in the book we studied, "Analytical Mechanics" by Grant R. Fowles and George L. Cassiday. This project required extensive knowledge of the geometry of an ellipse, analytical methods for modeling the gravitational potential, and numerical integration techniques to graph the gravitational potential. Overall, this project was a testimate of our ability to complex problems that when solved are not easy to prove correct and cannot be conveyed trivially.
